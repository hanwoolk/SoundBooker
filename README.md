# SoundBooker
============
> 개별 프로젝트

- 프로젝트 명 : SoundBooker (녹음 프로젝트 통합 관리 시스템)
- 인 원 : 1명
- 기 간 : 2023.03.20 ~ 2023. 03 .31
- DBMS : Oracle 11g Express Edition
- 언 어 : Java SE 9 Platform, HTML5, CSS, JavaScript
디자인 패턴 : MVC(Model 2)
오픈 Library : jQuery v3.4.1, jQueryUI, BootStrap은?
